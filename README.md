### 🤔 Apa itu Re-LMS-UNINDRA?

Pernah ngerasa tampilan LMS kampus agak kaku atau bikin mata cepet capek? **Re-LMS-UNINDRA** hadir sebagai *browser extension* (ekstensi peramban) *unofficial* yang gue desain khusus buat ngerombak UI/UX dari LMS Universitas Indraprasta PGRI (UNINDRA). 

Fokus utamanya gampang: **Bikin mata lebih adem, navigasi lebih cepet, dan ngerjain tugas jadi lebih *enjoy*!**

---

### ✨ Fitur Unggulan (Sneak Peek)

- 🎨 **Modern UI Redesign:** Tampilan antarmuka yang lebih *clean*, rapi, dan nggak kaku.
- 🌙 **Dark Mode Ready (Opsional):** Ngampus malam-malam nggak bikin mata perih lagi silau lihat layar putih.
- 🚀 **Navigasi Sat-Set:** Penempatan tombol dan menu dibikin lebih logis biar akses materi kuliah makin gampang.
- 📱 **Lebih Responsif:** *Layout* yang dioptimasi biar lebih enak dilihat di layar laptop ukuran apa aja.

### 📥 Cara Download & Install (Rilis Terbaru)

Karena ekstensi ini belum mejeng di Chrome Web Store, lo harus pasang secara manual via *Load Unpacked*. Tenang, gampang banget kok! Ikutin cara ini:

1. **Download File Ekstensi:**
   - Pergi ke halaman [Releases 📦](https://github.com/rizukurizu/Re-LMS-UNINDRA/releases).
   - Cari rilis versi paling atas (Latest), lalu *download* file `.zip` nya (misal: `Re-LMS-UNINDRA-v1.0.0.zip`).
   - Ekstrak file `.zip` tersebut di folder PC/Laptop yang gampang dicari.

2. **Pasang di Browser (Chrome / Edge / Brave):**
   - Buka browser kamu, lalu ketik ini di *address bar*: `chrome://extensions/` (atau `edge://extensions/` buat pengguna Edge).
   - Di pojok kanan atas, nyalain tombol **Developer mode** (*Mode Pengembang*).
   - Klik tombol **Load unpacked** (*Muat yang belum dikemas*) di kiri atas.
   - Pilih folder hasil ekstrak tadi, lalu klik **Select Folder**.
   - *Voila!* 🎉 Ekstensi udah aktif. Tinggal buka web LMS UNINDRA dan rasain bedanya!
